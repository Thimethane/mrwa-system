# Contributing to MRWA

Thank you for your interest in contributing to MRWA!

## Development Setup

1. Fork the repository
2. Clone your fork: `git clone https://github.com/Thimethane/mrwa-system.git`
3. Run setup: `./setup.sh`
4. Create a branch: `git checkout -b feature/your-feature`

## Code Standards

- **Python**: Follow PEP 8, use Black for formatting
- **JavaScript**: Use ES6+, Prettier for formatting
- **Testing**: Write tests for new features
- **Documentation**: Update docs for API changes

## Pull Request Process

1. Update tests
2. Update documentation
3. Run `make test` to ensure all tests pass
4. Submit PR with clear description
5. Wait for review

## Commit Messages

Use conventional commits:
- `feat:` New feature
- `fix:` Bug fix
- `docs:` Documentation
- `test:` Tests
- `chore:` Maintenance

Example: `feat: add PDF parsing support`

## Questions?

Open an issue or join discussions!
